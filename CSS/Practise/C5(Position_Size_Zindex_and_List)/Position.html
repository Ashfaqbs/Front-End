Position property is used to manipulate the position/location of the element following values
1. static - default and top , bottom , left , right , z-index has no effect the element is in default position is it will be in the default flow how its written it will be displayed like that
2. relative - top , bottom , left , right , z-index has effect the element will be in the normal flow but T , B ,L , R , -Z will work
3. absolute - element is removed from the normal flow and Positioned to 1st non static parent top , bottom , left , right , z-index has effect explained As we set the element position as absolute the element is removed from the normal flow , i.e the normal flow is header and its sub elements come ,then main > h1 > div > div> br > this text and article > its sub elements show render as they have added now this class box 3 position is absolute so what will happen is article will come , inside box1 , 2 will load as box 3 is absolute the elements will ignore it box 4 will take box3 flow position as box3 is now absolute and is out of flow now box 4 will be in 3 position but box3 follows a a position ie non static ancestor or parent , and now if we see its first parent is article and its position is static and next parent is main that is also static next is body itself that is also static and next is HTMl tag itself that is also static so it will follow Html position now we can use top , bottom , right, and left note : if the parent is non static , i.e relative , absolute , sticky etc the position of the child container is now absolute the T ,B , L ,R will change inside the parent container i,e the child container now will move inside the parent container T, B ,L, R will work with respect to non static parent/Ancestor and position of child container which is absolute will change w.r.t to the non static parent container
4. sticky - top , bottom , left , right , z-index has effect and positioned is fixed on the user scroll position
5. Fixed same as absoulte but the element is positioned to browser window i.e top , bottom , left , right , z-index has effect and if we scroll down it will be there if top , bottom , left , right , z-index is activated position is fixed
