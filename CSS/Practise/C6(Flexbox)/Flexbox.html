Flexbox aims to provide a better way to layout , align and distribute space among items in a container

flexbox or we also call it flex container and elements inside it we call flex elements here we have main axis i.e x axis and cross axis i.e y axis(need not to be always be vertical) main axis is in which direction our elements are rendering and cross axis is just perpendicular of it. display: flex; flex-direction: column-reverse or column or row-reverse or row to change in row wise or column wise and its row -> flex items in row or horizontal direction and row-reverse is reverse of the 1st in horizontal manner columnm -> flex items in vertical direction and ---||----

Properties which we use in container and not in flex items

*flex-direction - row ,column , and their reverse
*flex-wrap - wrap,wrap-reverse and no wrap as the screen size changes it can make the flex items to fit the screen by showing one or two flex items according to the the screen size , can adjust the flex items to fit the screen size useful for making responsive website
*Justify content - center , space around , space evenly and space between easiest way to center elements /* justify-content:center ; */ horizontal center
*align-items - vertical center but will have spaces between flex items
*align-content - vertical center but will not have spaces between flex items to see use flex-wrap: wrap and see both the Properties

Properties which we use in flex items and not in container

*Order - controls the order in which the items appears in flex container .box1 { border: 2px solid black; height: 90px; width: 90px; background-color: red; /* order: 4; */ } now 1st box will go to end
*align self -targeting individual flex items /* align-self: flex-end; */ on box1.
*flex grow - width must not ge given and now we ge give the flex grow will increase the size. .box4 { border: 2px solid black; height: 90px; /* width: 90px; */ flex-grow: 3; background-color: rgb(58, 4, 255); }
*flex shrink -width must not ge given and now we ge give the flex shrink will decrease the size.used when flex items overflow the container .box4 { border: 2px solid black; height: 90px; /* width: 90px; */ flex-shrink: 3; background-color: rgb(58, 4, 255); }
