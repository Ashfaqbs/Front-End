Mobile Devices:
Typically, mobile devices have a width less than or equal to 767 pixels. Therefore, you can use a media query with a max-width of 767 pixels to target mobile devices.
Tablets:

Tablets often have a width between 768 pixels and 1023 pixels. To target tablets, you can use a media query with a min-width of 768 pixels and a max-width of 1023 pixels.
Computers and Laptops:

Computers and laptops generally have wider screens. A commonly used width range for targeting these devices is 1024 pixels and above. You can use a media query with a min-width of 1024 pixels to apply styles specifically for computers and laptops.


/* Mobile devices */
@media only screen and (max-width: 767px) {
  /* Styles for mobile devices */
  /* For example: */
  body {
    font-size: 14px;
  }
  
  .container {
    width: 100%;
    padding: 10px;
  }
}

/* Tablets */
@media only screen and (min-width: 768px) and (max-width: 1023px) {
  /* Styles for tablets */
  /* For example: */
  body {
    font-size: 16px;
  }
  
  .container {
    width: 768px;
    margin: 0 auto;
    padding: 20px;
  }
}

/* Laptops and desktop computers */
@media only screen and (min-width: 1024px) {
  /* Styles for laptops and computers */
  /* For example: */
  body {
    font-size: 18px;
  }
  
  .container {
    width: 960px;
    margin: 0 auto;
    padding: 40px;
  }
}

In this example, media queries are used to adjust the font size and container width for different devices. Here's how it works:

Mobile devices: The max-width of 767px targets mobile devices. The font size is set to 14px, and the container width spans the entire screen width (width: 100%) with some padding.

Tablets: The min-width of 768px and max-width of 1023px target tablets. The font size is set to 16px, and the container width is fixed at 768px (width: 768px). The container is centered horizontally using margin: 0 auto, and it has some padding.

Laptops and desktop computers: The min-width of 1024px targets larger screens. The font size is set to 18px, and the container width is fixed at 960px (width: 960px). Similar to tablets, the container is centered and has padding.

These are just example values, and you can adjust the font sizes and container widths according to your specific design requirements.
