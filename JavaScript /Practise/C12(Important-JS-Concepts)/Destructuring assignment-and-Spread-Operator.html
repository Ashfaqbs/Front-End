Destructuring assignment and Spread Operator

Destructuring assignment is used to unpack values from an array ,or properties from objects into distinct variables Spread Operator allows an iterable such as an array or string to be expanded in places where zero or more arguments are expected , in an object literal , the spread syntax enumerates the properties of an object and add the key-value pairs to the objects being created 
    
        

            // without Destructuring
            let ar = [1, 2, 3, 4];
            let a = ar[0];
            console.log(a)
            // o/p
            // 1
    
    
           // with Destructuring
           //eg1
            let [a, b, c, d] = [1, 2, 3, 4]
            console.log(a)
            console.log(b)
            console.log(c)
            console.log(d)
            // o / p
            // 1
            // 2
            // 3
            // 4

            // eg2

            let arr = [1, 2, 3, 4]
            let [a, b, c, d] = arr;
            console.log(a)
            console.log(b)
            console.log(c)
            console.log(d)
            o / p

//to much data  then using ...Operator 
let arr = [1, 12, 3, 4, 5, 6, 7, 8, 9, 9, 1, 3]

let [a, b, ...restdata] = arr;
console.log(a, b, restdata)
