local and global Scope in JavaScript

    JavaScript has three types of scope and here local scope are block and function scope and global scope
  • Block scope
    • let and const provide block level scope which means that the variables declared inside a {} cannot be accessed from outside the block { let a =2; } console.log(a) //undefined as a is not available here
  • Function scope
    • Variables declared in the JavaScript Function become local to the Function //function scope function demo() { let ax = 9; console.log(ax) } demo() console.log(ax) //Uncaught ReferenceError: ax is not defined
  • global scope
    • A variable declared outside a Function becomes global //Global scope can be accessed in function and also in outside let g = 8; function demo1() { console.log(g) } demo1()//8 console.log(g)//8
