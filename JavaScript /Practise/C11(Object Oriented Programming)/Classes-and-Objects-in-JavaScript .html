Classes and Object 

In OOP a class is an extensible program-code template for creating objects , providing initial values for 
state(member variable) and implementation of behavior (member functions)


The basic syntax for writing a class is :

//class methods

Creating class , object , calling variable and functions



//class 1

        class Myclass {
            constructor() {
                console.log('from constructor')

            }

            //variables const/let no need to use
            a;
            b = 5;

            //functions no need to use function keyword , let ,const etc.. 
            doTask1(params) {
                console.log('dotask1')
            }

            doTask2(params) {
                console.log('dotask2')
            }


        }


        //creating object for the Myclass

        let object1 = new Myclass()

        let object2 = new Myclass()


        //calling variables and functions of the class
        console.log(object1.a)
        console.log(object1.b)

        console.log(object1.doTask1())
        console.log(object2.doTask1())


        console.log(object2.a)
        console.log(object2.b)
