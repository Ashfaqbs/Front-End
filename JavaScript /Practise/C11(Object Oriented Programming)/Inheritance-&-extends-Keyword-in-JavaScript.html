Inheritance & extends Keyword in JavaScript

    
        

Class Inheritance

Class Inheritance is a way for one class to extend another class, this is done by using extend keyword


The extend keyword

extends keyword is used to extend another class

Important
In JavaScript, when you define a derived class using the extends keyword, you are required to explicitly call the super() method within the constructor of the derived class. This is necessary because JavaScript does not automatically call the base class constructor for you.


In Java, the super() keyword is used implicitly to call the base class constructor. It is added by the compiler if you don't explicitly specify a call to the base class constructor using super() or this().

i.e we need to call super() or this keyword explicitly in JavaScript

//main class 1

class Fish {
    constructor() {

    }
    swim() {
        console.log('im swimming')
    }
}

class dolphin extends Fish {

    name
    constructor(name) {
        super();

        this.name = name
    }
    //dolphin specific method
    dolphinDance() {
        console.log('dolphin is dancing')
    }

}
class Shark extends Fish {
    name
    constructor(name) {
        super();
        this.name = name
    }

    //Shark specific method
    sharklikeHunting() {
        console.log('Shark  is hunting')
    }

}


let dol = new dolphin('dolphin');
let shark = new Shark('sharky');
dol.swim()
dol.dolphinDance()
shark.swim()
shark.sharklikeHunting()
