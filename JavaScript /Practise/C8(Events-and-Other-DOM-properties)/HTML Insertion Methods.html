Insertion of a Node inside the DOM

    Some insertions methods are 1st create and add data/text to it and insert by this functions
  • node.append(e)
    • append at the end of the node
  • node.prepend
    • insert at the beginning of the node
  • node.before(e)
    • Insert before node
  • node.after(e)
    • Insert after the node
  • node.replaceWith(e)
    • Replace node with the given node
