for this button we have not used onclick to provide functions as it can take only one function , but its is calling two functions for same actions for now i.e click and we were able to call two functions by addEventListener 
        let btn = document.getElementById('btn')

removeEventListener

removing the event with some conditions, Note: try to pass the functions as variable and not as the same whole fictions as it will be treated as different function object 

        let num = Number.parseInt(prompt('Please enter the num'))
        let btn2 = document.getElementById('btn2')


        let a = () => {
            alert('hello world')
        }

        let b = () => {

            alert('hello world2')

        }
        btn2.addEventListener('click', a);
        btn2.addEventListener('click', b);

        if(num===2)
        {
            btn2.removeEventListener('click', b);
        }
