• Child Nodes

    Direct children for the element are children eg: head and body are child nodes for html tag, here head and body are brother/sisters or can be called siblings

  • Descendant Nodes

    all nested element, children , their children are are Descendant nodes

First child , Last child and child nodes

  • element.first child
    • First child element
  • element.last child
    • Last child element
  • element.child nodes
    • All child nodes 
    console.log(document.body.firstChild)//op 
     ...

    console.log(document.body.lastChild)//op script tag
    console.log(document.body.childNodes)//op [text, ul, text, h1, text, ul, text, h1, text, h1, text, script]
