Difference between var , let and const

  • VAR
    • Before ES6 var was used , but nowadays no one uses it ,as it will cause bugs in the applications and we should avoid it
  • Const
    • used to declare constants and for that variable we cannot change values and this is blocked scope
  • let
    • We use let to declare block scope variables
Note:
block means if block , function block , or a empty block etc
